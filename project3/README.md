# Project #3: SIC/XE LINKING LOADER

## HOW TO USE
* Compile and executing the file
  ```bash
	$> make
	$> ./20171635.out
  ```
## EXAMPLE FOR LINKING PROJECT3
* progaddr
  ```bash
  	sicsim> progaddr 4000
	Program starting address set to 0x4000.
  ```

* linking loader
  ```bash
  	sicsim> loader proga.obj progb.obj progc.obj
  ```
